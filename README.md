# ЛИС для нефтебазы (Streamlit Cloud)

## Интерактивное веб-приложение на **Streamlit**, предназначенное для автоматизации **отбора и анализа проб нефти на нефтебазе**.  

**Функционал:**

- Приём данных о вагонах и цистернах.
- Расчёт количества проб по ГОСТ и позиции отбора.
- Проверка нормативов качества: плотность, вязкость, вода, соли, механические примеси, сера, температура вспышки и кипения.
- Автоматическое определение годности пробы: **ГОДНА / УСЛОВНО ГОДНА / НЕГОДНА**.
- Генерация **QR-кода** для каждой пробы.
- Генерация **PDF-протокола** через `FPDF2`.
- Авто-дэшборд по пробам с подсветкой несоответствий.
- Хранение данных в **Supabase** через REST API.
- Уникальность пробы проверяется по `(train_number, tank_number)`.

---

## Технологии

- **Python 3.14**
- **Streamlit** — интерфейс и дэшборд
- **Supabase** — база данных (PostgreSQL)
- **FPDF2** — генерация PDF без внешних зависимостей
- **qrcode + Pillow** — QR-коды для каждой пробы
- **requests** — работа с REST API Supabase
-  **проект полностью совместим с **Streamlit Cloud** и не требует компиляции C-библиотек.**

## Установка и запуск

1. Клонируем репозиторий:

```bash
git clone https://github.com/yourusername/oil-lis.git
cd oil-lis
```
2. Устанавливаем зависимости:
```bash
pip install -r requirements.txt
```
3. Запускаем локально 
```bash
streamlit run streamlit_app.py
```
## Настройка Supabase
1. Создать базу данных в Supabase.
2. Создать таблицу samples с необходимыми для вас колонками.
3. Получить SUPABASE_URL и SUPABASE_KEY, добавить в secrets.toml Streamlit Cloud:

```TOML
SUPABASE_URL = "https://xxxx.supabase.co"
SUPABASE_KEY = "public-anon-key"
```
## Использование
- *Вводим данные о составе, цистерне и пробе.*
- *Система автоматически проверяет нормативы.*
- *Генерируется QR-код для пробы.*
- *PDF-протокол можно скачать кнопкой.*
- *Все данные сохраняются в Supabase и выводятся в авто-дэшборде.*
